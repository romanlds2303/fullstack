#######
Pasos para compartir un Proyecto en GitHub
#######

1Un miembro del equipo crea un proyecto y un repo.

2Agrega colaboradores en:

        
Settings
Colaboradores
Add people con el email del compañero/usuario de GitHub

3Realiza un git clone en la terminal ....url del proyecto

    git clone https://github.com/BerniPas/repocompartir.git

4Desarrollamos código

5Realizamos un git pull => verifica si hay cambios en el repo remoto y los descarga.

6Realizo un git push .. url del repo
