#######
Pasos para compartir un Proyecto en GitHub
#######

Un miembro del equipo crea un proyecto y un repo.

Agrega colaboradores en:

        
Settings
Colaboradores
Add people con el email del compañero/usuario de GitHub

Realiza un git clone en la terminal ....url del proyecto

    git clone https://github.com/BerniPas/repocompartir.git

Desarrollamos código

Realizamos un git pull => verifica si hay cambios en el repo remoto y los descarga.

Realizo un git push .. url del repo
